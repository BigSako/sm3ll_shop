Welcome to SM3LL Panda Delivery!

This is an application which allows you to order ingame items for EVE Online with ingame ISK (aka: Jump Freighter Delivery Service). The application is currently only available for Northern Coalition., and uses the EVE Online Single Sign-On (SSO) and authenticated CREST access.
The EVE Single Sign On (SSO) is a system put into place by CCP itself, to allow secure authentication against 3rd-party sites like this one. If you don’t trust this site with your username and password, then log into one of the official CCP sites first (e.g., EVE Gate, Forums) and on that login page, click the checkbox "remember me". That way, when you come back here, you will not be prompted for your username and password, but only for the character to use.

NOTE: When setting up, please make sure you use a character that is currently in Northern Coalition., else the application will not allow you to order anything.

This app depends on Python Social Auth to integrate EVE SSO with Django's builtin user authentication system and uses PyCrest to interact with the CREST API.

Use the Login button below to authenticate! It will redirect you to the EVE Online Single Sign On (SSO). After logging in, you will be redirected back to this site.
